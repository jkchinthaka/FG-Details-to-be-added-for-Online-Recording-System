# Usability test script

Status: **HUMAN_APPROVAL_REQUIRED** — do not mark complete without signatures.

## Setup

- UAT environment only  
- Real role accounts (no shared passwords in the report)  
- Devices: desktop, tablet, production phone  
- Optional slow-network throttle  

## Tasks

### FG_OPERATOR

1. Sign in and (if prompted) change password  
2. Create a freezer-truck or cleaning record  
3. Add evidence and submit  
4. Recover from airplane-mode interruption if available  

### FG_SUPERVISOR

1. Open To Check  
2. Open oldest pending item  
3. Reject with reason  
4. Return to queue without losing place  

### QA_EXECUTIVE

1. Open To Verify  
2. Review evidence  
3. Verify or reject  
4. Open related corrective action  

### FOOD_SAFETY_TEAM_LEADER

1. Oversight from Home  
2. Escalate or verify a high-risk item  
3. Open Reports  

### AUDITOR

1. Navigate reports read-only  
2. Export (if permitted)  
3. Confirm no write actions appear  

### SYSTEM_ADMINISTRATOR

1. Open users admin  
2. Preview a checklist template  
3. Confirm preview banner and reset confirmation
