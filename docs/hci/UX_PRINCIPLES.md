# UX principles

1. Visibility of system status  
2. Match real factory language  
3. User control and freedom (cancel, logout, undo where safe)  
4. Consistency with Nelna patterns  
5. Error prevention before error messages  
6. Recognition over recall  
7. Flexibility for expert and novice operators  
8. Aesthetic and minimal — no decorative clutter on floor devices  
9. Help users recover from errors  
10. Documentation for support, not for every tap  

Operational priority: speed, clarity, and safety over visual novelty.
