# Project brief — Nelna FG Digital Recording System

## Product name

Nelna FG Digital Recording System

## Owner / developer

Chinthaka Jayaweera

## Goal

Digitise Finished Goods and Quality Assurance recording at Nelna Farm with a fast, mobile-first Progressive Web App that replaces paper records.

## Device priority

1. Mobile phone
2. Tablet
3. Desktop

## UX target

A normal daily record must be completable in approximately 3–5 actions using an exception-based pattern:

1. Open assigned task
2. Mark All Acceptable
3. Modify only failed items
4. Submit

## Core roles

- FG Operator
- FG Supervisor
- QA Executive
- Food Safety Team Leader
- System Administrator
- Auditor

## Non-goals (foundation phase)

- Full production auth and master-data administration (later prompts)
- Fake production integrations
- Generic admin dashboard templates unrelated to Nelna factory operations

## Success measures

- Operators can record without typing on the happy path
- Failed items capture structured evidence
- Historical template versions remain stable
- Verified records are never physically deleted
